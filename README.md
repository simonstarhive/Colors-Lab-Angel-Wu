# COLORS LAMP Application

## Description
COLORS is a web application built using the LAMP stack that allows users to:
- Log in
- Add colors to their account
- Search for stored colors

## Technologies Used
- Linux (Ubuntu)
- Apache
- MySQL
- PHP
- HTML
- CSS
- JavaScript

## Project Structure
api/ - PHP backend API endpoints  
public/ - Frontend files (HTML, CSS, JS)

## Setup Instructions
1. Install LAMP stack
2. Create MySQL database COP4331
3. Import tables (Users, Colors, Contacts)
4. Configure database credentials in API files
5. Place public folder contents in /var/www/html
6. Access via http://simoncitopage.shop/

## Running the Application
Visit:
http://simoncitopage.shop/

Use Postman or frontend UI to test API endpoints.

## Assumptions & Limitations
- Password hashing uses MD5 (for class purposes only)
- No HTTPS configured
- Minimal input validation
